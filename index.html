HOW TO PLAY

1 Start Playing The harmonium loads automatically. Press keys on your keyboard or click the on-screen keys to play notes.

2 Learn the Keys White keys map to S, `, Q–P, [, ], \, and ;. Black keys use A, 1, 2, 4, 5, 7, 8, 9, -, =, and '. The visual keyboard shows which notes you're playing.

3 Customize Sound Use the controls to add reverb, change octaves, transpose, or stack reeds for different sounds.

This engine uses Subtractive Synthesis to mimic physical reeds. Blow notes (exhale) are mapped to the top row, while Draw notes (inhale) are mapped to the home row to allow for rapid chord transitions.
